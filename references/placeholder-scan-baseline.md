## 典型错误模式

### ug (维吾尔语) — 空格拆散占位符
```
源: 序列号：%s
译: تەرتىپ نومۇرى:% s    ← % s 带空格，Android 无法识别
```

### vi (越南语) — 占位符整体丢失
```
源: "{0}"的"{1}"完成灌溉
译: Hoàn thành tưới        ← {0} 和 {1} 都丢了
```

### es-ES — {N} 被改成 %N$s
```
源: "{0}"的"{1}"水压为 {2} 千帕
译: La presión de "%1$s" en "%2$s" es de %3$s kPa   ← 格式体系换了
```

### th (泰语) — 数字占位符被翻译成文字
```
源: %1$s水位低于%2$s自动开泵
译: ปั๊มจะเริ่มทำงานเมื่อระดับน้ำต่ำกว่า 1%  ← %2$s 变成 "1%"
```
